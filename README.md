# Team Task Manager

##  Features

* User Authentication (JWT)
* Admin & Member roles
* Create & assign tasks
* Task status tracking (Pending, In Progress, Completed)
* Dashboard with live stats

##  Tech Stack

* Frontend: React (Vite)
* Backend: Node.js + Express
* Database: MongoDB Atlas

##  Project Structure

* client → React frontend
* server → Express backend

##  How to Run

### Backend

cd server
npm install
npm run dev

### Frontend

cd client
npm install
npm run dev
