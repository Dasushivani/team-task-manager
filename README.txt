Project Overview:
Team Task Manager is a full-stack web application designed to help teams efficiently manage tasks, assignments, and workflow. It includes secure authentication, role-based access, dashboard analytics, task creation, task assignment, and task status tracking.

---

Features:

* User Login Authentication (JWT)
* Role-Based Access (Admin / Team Member)
* Admin Dashboard
* Task Creation
* Task Assignment
* Task List View
* Task Status Update (Pending / In Progress / Completed)
* Logout Functionality
* MongoDB Atlas Database Integration
* Fully Deployed Full-Stack Application

---

Tech Stack:

Frontend:

* React (Vite)
* React Router DOM

* Node.js

---

How to Run Locally:

1. Clone the repository:
   git clone https://github.com/Dasushivani/team-task-manager.git

2. Backend Setup:
   cd server
   npm install
   npm run dev

3. Frontend Setup:
   cd client
   npm install
   npm run dev

---

Project Workflow:

1. User logs in with Admin or Team Member credentials
2. Admin accesses dashboard analytics
3. Admin creates and assigns tasks
4. Tasks are displayed in task list
5. Team members can view assigned tasks
6. Task statuses can be updated
7. Users can securely logout
